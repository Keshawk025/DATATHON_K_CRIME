## 📋 Project Overview

**K-CRIME** is an advanced, AI-powered crime analytics and visualization platform built specifically for the Karnataka State Police. It transforms fragmented First Information Reports (FIRs) and raw crime records into an interactive, highly visual administrative dashboard.

The system helps commanding officers, investigators, and intelligence analysts detect crime hotspots, identify repeat offenders, analyze sprawling criminal networks, and predict future risk areas using state-of-the-art machine learning.

---

## 🚀 Key Features

*   🌍 **Geospatial Intelligence (Heatmaps):** Interactive Leaflet maps rendering crime density, district-level risk scores, and high-risk hotspots.
*   📊 **Live Analytics Dashboard:** Real-time KPI tracking, category distributions, severity analysis, and district rankings powered by dynamic SQL aggregation.
*   🕸️ **Criminal Network Analysis:** Cytoscape.js integration to visually map connections between perpetrators, victims, shared addresses, and associated vehicles.
*   🤖 **CrimeMind AI Copilot:** A Google Gemini-powered chat assistant that allows investigators to query the database and ask for case summaries using natural language.
*   🔮 **Predictive Intelligence:** Scikit-learn integrated models forecasting monthly crime volumes and identifying anomalous behavior spikes.

## 💻 Technology Stack

### Frontend (User Interface)
- **Framework:** React 19 + TypeScript (built via Vite)
- **Styling:** Tailwind CSS (v4) + Shadcn/UI (Strict Government-Standard UI styling)
- **State Management:** TanStack Query (React Query)
- **Visualizations:** Recharts, Cytoscape.js, Leaflet.js
- **Routing:** React Router DOM

### Backend (API & AI Layer)
- **Framework:** FastAPI (Python) + Uvicorn
- **Database:** SQLite (local development) / PostgreSQL (production ready)
- **ORM:** SQLAlchemy + Alembic (migrations)
- **Authentication:** JWT (JSON Web Tokens) with Passlib & Bcrypt
- **Machine Learning:** Pandas, NumPy, Scikit-learn
- **Generative AI:** Google Generative AI (Gemini 1.5)
